# StataAgent 项目背景

## 项目简介

StataAgent 是一个自动化实证分析智能体系统。用户输入实证研究主题后，Agent 自动完成从变量定义、数据清洗到 Stata 回归分析的完整实证研究流程。

## 核心功能

1. **需求解析** - 根据用户输入的实证主题，构思变量定义和数据需求
2. **数据清洗** - 自动处理用户上传的多源数据，完成清洗与合并
3. **实证分析** - 自动生成 Stata 代码并执行回归分析，输出统计结果

## 工作流程

```
用户输入主题 → 变量定义 → 用户上传数据 → 数据清洗合并 → 描述性统计 → 回归分析
```

## 技术栈

- Python (数据处理与 Agent 逻辑)
- 必须使用UV进行环境依赖管理（环境管理）
- Stata (计量回归分析)
